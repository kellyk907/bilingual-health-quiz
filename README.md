# Bilingual Public Health Quiz – English / Español

Live → https://bilingual-health-quiz.vercel.app

A React + Tailwind + i18next app that teaches CDC health facts in **English and Spanish**.

Built in 48 hrs to prove I ship **accessible, bilingual, production-ready apps**.

## Tech
- React 18 + Vite
- Tailwind CSS
- i18next (real-time language toggle)
- Vercel (1-click deploy)

## Features
- 5 real CDC questions
- Progress bar + score
- Mobile-first, keyboard accessible
- ARIA labels
